## 3. Детальное описание методов

### Глава 1: Методы (1-5)

#### 1. fraction() - Вычисление дробной части числа
Принцип работы: 
- Запрашивает число типа double
- Вычисляет дробную часть: result = number - (int)number
- Пример: Ввод: 5.25 → Вывод: 0.25

#### 2. charToNum() - Преобразование символа в число
Принцип работы:
- Считывает один символ
- Проверяет цифра ли это (0-9)
- Преобразует через разность кодов: result = inputChar - '0'
- Пример: Ввод: '7' → Вывод: 7

#### 3. isTwoDigits() - Проверка на двузначное число  
Принцип работы:
- Проверяет диапазон: number >= 10 && number <= 99
- Пример: 25 → true, 5 → false, 100 → false

#### 4. isInRange() - Проверка вхождения в диапазон
Принцип работы:
- Принимает границы диапазона и число
- Определяет min и max границы
- Проверяет: num >= min && num <= max
- Работает с любым порядком границ

#### 5. isEqual() - Проверка равенства трех чисел
Принцип работы:
- Сравнивает три числа: a == b && b == c
- Пример: 3,3,3 → true, 2,2,5 → false

### Глава 2: Условия (6-10)

#### 6. abs() - Модуль числа
Принцип работы:
- Возвращает -x если отрицательное, x если положительное
- Использует: x < 0 ? -x : x
- Пример: -7 → 7, 5 → 5

#### 7. is35() - Делимость на 3 или 5
Принцип работы:
- Проверяет: x % 3 == 0 и x % 5 == 0
- Возвращает true если делится на 3 ИЛИ на 5, но НЕ на оба
- Логика: (div3 || div5) && !(div3 && div5)
- Пример: 15 → false, 10 → true, 9 → true

#### 8. max3() - Максимум из трех чисел
Алгоритм:
1. max = x
2. if (y > max) max = y  
3. if (z > max) max = z

#### 9. sum2() - Особенная сумма
Условие:
- Если сумма в диапазоне 10-19 → возвращает 20
- Иначе → возвращает actual сумму
- Логика: (sum >= 10 && sum <= 19) ? 20 : sum

#### 10. day() - День недели
Принцип работы:
- Использует switch-case для преобразования числа в день
- 1-понедельник, 7-воскресенье
- default: "это не день недели"

### Глава 3: Циклы (11-15)

#### 11. listNums() - Последовательность чисел
Алгоритм:
- Цикл от 0 до X включительно
- Конкатенация чисел в строку с пробелами
- Пример: 5 → "0 1 2 3 4 5"

#### 12. chet() - Четные числа
Особенность:
- Использует шаг 2: i += 2
- Более эффективно чем i % 2 == 0
- Пример: 9 → "0 2 4 6 8"

#### 13. numLen() - Количество цифр
Алгоритм:
- Делит число на 10 пока не станет 0
- Считает количество делений
- Пример: 12567 → 5 итераций

#### 14. square() - Квадрат из звездочек
Принцип:
- Двойной вложенный цикл
- Внешний: строки, Внутренний: столбцы
- Печатает '*' в каждой позиции
- Пример: 3 → 3x3 квадрат

#### 15. rightTriangle() - Правый треугольник
Алгоритм:
- Первый цикл: строки
- Второй цикл: пробелы (убывающе)
- Третий цикл: звездочки (возрастающе)

### Глава 4: Массивы (16-20)

#### 16. findFirst() - Первое вхождение
Алгоритм:
- Линейный поиск по массиву
- Возвращает первый найденный индекс
- Возвращает -1 если не найдено

#### 17. maxAbs() - Максимальный по модулю
Особенность:
- Сравнивает абсолютные значения
- Сохраняет оригинальное значение (со знаком)
- Ручное вычисление модуля без Math.abs()

#### 18. add() - Вставка массива
Алгоритм:
1. Копирует элементы до позиции вставки
2. Вставляет новый массив  
3. Копирует оставшиеся элементы
- Пример: [1,2,3] + [4,5] на pos=1 → [1,4,5,2,3]

#### 19. reverseBack() - Реверс массива
Принцип:
- Создает новый массив
- Заполняет в обратном порядке: result[i] = arr[arr.length-1-i]
- Пример: [1,2,3] → [3,2,1]

#### 20. findAll() - Все вхождения
Двухпроходный алгоритм:
1. Первый проход: подсчет количества вхождений
2. Создание массива нужного размера
3. Второй проход: заполнение индексами
- Пример: [1,2,3,2,5], 2 → [1,3]
